Instrucciones: Para correr el programa se debe de ejecutar la clase "VistaPrincipal", la cual se encuentra en la carpeta src en el paquete modelo al inicio de la ejecucion se solicita que se ingrese una medida de un lado, una vez ingresada aparecera un menu como el siguiente:
			****Elija una opcion****
			
			  1.-Perimetro
			  2.-Area
			  3.-Volumen
			  4.-Cancelar
  		  	  5.-Salir

Para saber el perimetro se oprime el número 1
Para saber el área se oprime el número 2
Para saber el volumen se oprime el número 3
En caso de que se quiera cambiar la medida ingresada inicialmente se oprime el 4 y vuelve a preguntar la medida del lado
Para salir del programa se oprime el numero 5
Una vez elegido una opcion dentro de las opciones 1 al 3 se muestra  un menu interno:
			****Elija una opcion****
			   1.-Volver al menu principal
			   2.-Nueva medida para el lado
El cual nos permite regresar al menú principal si asi se desea o ingresar una nueva medida al lado
